# 設定ファイル
**/etc/profile**  
全ユーザー共通の初期環境定義  
ログイン時に一度だけ実行

グローバルPATH（環境変数）  
LANG / LC_*  
umask  
/etc/profile.d/*.shの読み込み

<br>

**~/.bash_profile（ユーザー個別・ログイン時）**  
ユーザー単位のログイン初期化  
環境変数の最終調整  

ユーザー専用PATH  
EDITOR / PAGER  
ssh-agent起動  
~/.bashrcの読み込み

<br>

**etc/bashrc**  
全ユーザー共通の対話bash設定  

共通エイリアス
共通補完設定

<br>

**~/bashrc**  
ユーザー個別の対話bash設定  
bash対話操作性  
bashを起動するたびに実行

エイリアス
シェルオプション
プロンプト（PS1）
関数定義

<br>


# ファイルの読み込み順序
コンソールログイン、SSHログイン、``su -``でログインすると設定ファイルが読み込まれる


1. /etc/profile
2. ~/.bash_profile
   ＊以下の順に存在したものを実行  
   ・ ~/.bash_profile  
   ・ ~/.bash_login  
   ・ ~/.profile  
3. 上記のファイルに記載されている場合  
   /etc/.bashrc、~/.bashrc、/etc/bash.bashrc等が呼び出される
